# Spesa Counter

## How to build
- Install npm
- Clone the repo and open the terminal
- Run **npm install --dev** in the terminal
- Run **npm run build** to build the project once
- Run **npm start** to start the live reload server
- Connect to **localhost:8080** via browser

## How to publish
